# Digitalizador de Tabla Operatoria 🏥

## Descripción del Proyecto
Esta aplicación web está diseñada para optimizar y modernizar la visualización de la tabla operatoria diaria en Clínica MEDS. 

El objetivo principal es recibir fotografías de la programación quirúrgica (las cuales suelen presentar reflejos, ángulos distorsionados, o mezclar texto impreso en pantalla con anotaciones manuscritas) y transformarlas en un formato digital estructurado, claro y accesible desde cualquier dispositivo.

## Características Principales
* **Preprocesamiento de Imágenes:** Limpieza y ajuste de contraste de fotografías para mejorar la legibilidad.
* **Reconocimiento Óptico de Caracteres (OCR):** Extracción automatizada de texto impreso y escritura a mano.
* **Interfaz Web Intuitiva:** Plataforma accesible 24/7 para el equipo médico, sin necesidad de instalaciones adicionales.
* **Presentación Estructurada:** Conversión de los datos crudos a una tabla interactiva y limpia.

## Tecnologías Utilizadas
* **Lenguaje Base:** Python
* **Frontend y Despliegue:** Streamlit (Streamlit Community Cloud)
* **Procesamiento y Visión:** OpenCV y herramientas de OCR (Tesseract / APIs de Visión)
* **Control de Versiones:** Git y GitHub
